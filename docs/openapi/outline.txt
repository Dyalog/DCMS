OpenAPI

Dyalog Content Management System
==============================
v4.0.0

CRUD and admin endpoints are authenticated and require an API key. All other endpoints are public and do not require authentication.

CRUD /path is short for:
GET /path
POST /path
GET /person/{id}?columns=[{col1},{col2}]
PUT /path/{id}
DELETE /path/{id}
[{  // of those objects affected by the CRUD operation
    id: 0,
    // Other fields depend on the specific table
}]

GET /version
"4.0.0"
GET /presenters
["name1", "name2", ...]
GET /events?has_videos=true
[{
    id: 0,
    title: "",
    type: "",
    end_date: ""    // "YYYY-MM-DD"
    start_date: "", // "YYYY-MM-DD"
    url_slug: "",
}]
GET /videos
[{
    category: "",
    description: "",
    event: "",
    event_shortname: "",
    presented_at: "", // Year and month of presentation as "YYYY-MM-DD hh:mm:ss"
    presenter: "",    // Comma-separated list of presenter names
    published_at: "", // YouTube video publish date as "YYYY-MM-DD hh:mm:ss"
    thumbnail: "",
    title: "",
    url: "",
    youtube_id: ""
}]
GET /videos/{youtube_id}
{GET /videos}
GET /videos/{youtube_id}/recommended

GET /videos/?event_shortname={shortname}&presenter=[{id1},{id2}]&search={"searchterms"}
[{GET /videos}]
            &per_page={n}&page={n}
{
    path: "",     // The URL for this endpoint
    per_page: 0,
    current_page: 0,
    data: [{GET /videos}],
    from: 0,      // Ordinal number of the first result on this page
    to: 0,        // Ordinal number of the last result on this page
    total: 0,     // Total count of results for this query

    last_page: 0,

    last_page_url: "",
    links: [{   // Link objects describe numbered, previous and next buttons used on paginated results pages
        active: false,
        label: "",
        url: ""
    }],
}
